Primeros pasos con grid

Grid nos permite principalmente trabajar el layout en forma de rejilla. Es una poderosa opción que ofrece CSS para estructurar los elementos de un sitio web.

Lo que buscaremos en este corto pero profundo curso es profundizar en el uso de los grids y como podemos usarlo como alternativa a herramientas como bootstrap (aún te quiero 💗)

medida repeat()

Como vimos, un grid consta de columnas y filas y puede darse el caso en el que tengamos que indicar la misma medida para varias columnas, lo cual hacerlo a mano es más lento.

#1
grid-template-columns: 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr;
En el ejemplo anterior vemos que hay 12 columnas con medidas 1fr

Si bien se puede expresar de la manera anterior, esto hace que el código sea poco legible y muy largo. Para evitar esto tenemos el uso de la medida repeat()

El comportamiento es como si fuera una función, le enviamos como primer argumento las veces que se va a repetir la medida y como segundo argumento la medida que se repetira. De esta forma el código anterior se indicaría de manera más sencila de la siguiente forma:

#2
grid-template-columns: repeat( 12 , 1fr )

Las dos instrucciones son equivalentes pero la segunda es mucho más legible y más fácil de comprender.

Grid básico
display: grid
grid-template-columns: repeat( 12 , 1fr )

Medidas Especiales

Para determinar el tamaño de cada uno de los raíles que vamos a determinar, hacemos uso de las medidas estándar px em rem % pt. Pero en los grids también contamos con medidas especiales que nos serviran para crear diseños mucho más dinámicos.

Medida fr

La medida fr es una medida especial que nos permite controlar el tamaño de un elemento sin tener una medida Estática. Estas medidas son utilizadas por las insrucciones grid-templete-columns y grid-templete-row para definir los espacios de los raíles.

Instrucciones para los grid-items

grid-row y grid-column

Una de las caracteristicas de grid layout es que podemos controlar zonas dentro del grid.

Las zonas son grupos de grids que nos permiten agruparlos y que se comporten como un solo elemento.

Esta instrucción permite indicar con las lineas del grid, las dimensiones del area.

grid-column

Esta propiedad permite indicar en el grid desde que linea de la columna hasta que linea de la columna formará el ancho del area.

Por lo tanto, si tenemos una instrucción:

grid-column: 1/3

Indicará que ese elemento irá desde la linea 1 hasta la linea 3 y lo mismo para la instrucción grid-row, solo que esta indicará las lineas del row.

¿Qué es grid implicito?

Supongamos que creamos un grid que tiene un grid-template-columns: 1fr 1fr y un grid-template-rows: 100px 100px

Estamos hablando de un grid de 2x3 con sus respectivas dimensiones, pero que ocurre cuando en vez de 4 items tenemos 6.

Para manejar estos elementos hacemos uso de las propiedades grid-auto-rows grid-auto-columns y grid-auto-flow

grid-auto-flow

Antes de indicar el manejo de las dimensiones de los items implicitos, hay que hablar de la dirección de los nuevos elementos.

Por defecto grid, al momento de crear nuevos elementos dentro del layout los posiciona en filas row poniendo el nuevo carril justo debajo del anterior. Al momento de que hayan items implicitos estos iran poniendose debajo de los elementos explicitos.

Para cambiar la dirección de los mismos hacemos uso de grid-auto-flow el cual indica la orientación de los elementos implicitos.

Grid Dinámico

Una de las ventajas de grid es que podemos hacer diseños responsives sin necesidad de usar mediaquerys lo cual nos permite realizar diseños complejos con menos lineas de código y mucho más ordenado.

La función repeat() que se usa en los grids, es una función que nos permite ingresar palabras clave que nos proporcionará comportamientos que ayudarán la diseño a ser mas repsonsivo.

Los valores explicados a continuación solo se pueden usar dentro de la función repeat() que esta a su vez solo se usa en las propiedades grid-template-rows, grid-template-columns, grid-auto-rows y grid-auto-columns.

Valor min-content

Este valor indica a la columna o a la fila del grid que será tan ancho o alto como lo sea la cadena o elemento más pequeño que la contenga.

Grid con repeat(3, min-content)

Valor max-content

Este valor es la contraparte del valor min-content indicando este que la columna será tan larga como el elemento más largo encontrado en el mismo

Una de las particularidades es que en el valor min-content buscará el elemento más pequeño como por ejemplo, en una cadena de caractéres, buscará la palabra más larga dentro de esta, y será esta palabra la que indicará el ancho de la columna

En el caso del valor max-content, este usará la cadena entera de caractéres sin hacer brake-line en los textos

minmax() es una función de CSS que nos permite indicar el valor mínimo y máximo que va a tener una columna para hacerla responsive. A esta función se le pasan 2 valores minmax(dimensión_mínima,dimensión_máxima).

Este valor indicará a la columna/fila que reparta el espacio entre los valores mínimo y máximo

valor repeat(auto-fill,minmax(xx,xx))

Este valor se usa en el primer argumento de la función repeat() y este nos permite modificar el flow de los grid items

Cuando trabajamos con la funcion repeat() anteriormente, le enviabamos como primer argumento un INT indicando a cuantas columnas iba a afectar. Pero que ocurre si queremos que al haber pantallas más grandes, vaya acomodando los elementos con propiedades determinadas?

Para solucionar ello utilizamos la palabra clave auto-fill. Esta palabra clave indica al grid que si en el espacio que ocupa el grid pueden caber elementos unos al lado de otros los vaya acomodando.

Grid con auto-fill
#grid-auto-fill {
    grid-template-columns: repeat(auto-fill, minmax(50px, 1fr));
    grid-auto-rows: 50px;
}

Alineación y control de flujo

En esta sección vamos a ver como se puede controlar la alineación y control de flujo dentro de un grid.

Instrucciones para los items del grid

Estas instrucciones afectan a todos los hijos del grid por igual, permitiendo darles valores para modificar su layout en los ejes vertical y horizontal.

Estas instrucciones modifican al conjunto de elementos que forman parte del grid (grid-items).

Instrucciones Eje Horizontal

justify-items

La propiedad justify-items nos permite controlar el flujo del eje horizontal de los elementos del grid. Al aplicar estos valores los grid items internos serán tan anchos como lo indique su max-content en el caso de que no tenga las propiedades width y height.

  • stretch

    Este valor es la que vienen por defecto los grid items y estas ocupan todo el espacio disponible dentro de la columna indicada.

  • start

    Este valor posiciona el contenido lo mas a la izquierda (start)

  • Center

    Este valor centra los items.

  • end

    Este valor posiciona a los elementos lo más bajo del espacio asignado.

Instrucciones para el eje vertical

align-items

Esta instrucción alinia los items de forma vertical permitiendo controlar su disposición en el layout del grid. Comparte las mismas características que justify-items respecto a las dimensiones del grid.

  • stretch

    Este valor es la que vienen por defecto los grid items y estas ocupan todo el espacio disponible dentro de la fila indicada.

  • start

    Este valor posiciona en lo mas alto del espacio asignado.

  • Center

    Este valor

  • end

    Este valor posiciona a los elementos a la derecha del grid (end)

Instrucciones para el grid container

Este conjunto de instrucciones nos permizten modificar las dimensiones y el layout del grid-container modificando sus rows y sus columns.

Los instrucciones son semanticamente similares ya que solo deberemos cambiar xxx-items por xxx-content y en comportamiento son similares.

Alineación Horizontal

justify-content

Con esta propiedad podemos modificar el comportamiento de las columnas del grid. Esto se aprecia principalmente en layouts con columnas de unidades fijas, no se va a apreciar correctamente si estamos trabajando con la medida fr

Esta propiedad nos permite, al igual y con los mismos valores de la instrucción justify-items, modificar la posición en el eje horizontal de las columnas.

Como se ha indicado antes, esta propiedad recibe los mismos valores que las instrucciones para los items start center end. Pero esta incluye otros valores como los son space-around space-between space-evenly. Estos valores comparten el mismo comportamiento que tienen en las propiedades flex.

Alineación Vertical

align-content

para resumir, es la propiedad que nos permite modificar la posición de los elementos del grid en su eje vertical.

Este comparte los mismos valores que justify-content.

Controladores eje horizontal
justify-content: start
Controladores eje vertical
align-content: start
